Pasos para el desarrollo del portafolio:

- Estructurar un maquetado del sitio
- Buscar video utilizado en el index de la página con los formatos mp4, webm, ogv
- Crear carpeta portafolio y archivo index.html, y también las carpetas src/img, src/js, src/css y video
- Generar archivo package.json con comando: npm init
- Instalar dependecias del proyecto: npm install --save-dev sass gulp gulp-sass
- Iniciar con el código semantico HTML
- Estructura general HTML 
- Añadir fuentes Google utilizadas en el proyecto
- Añadir etiquetas HTML header y seccion para las secciones más importantes de la página
- Añadir archivos scss globales, index, mixins, normalize y variables, que fueron utilziados en el proyecto anterior
- Añadir archivo gulpfile.js con la configuración de compilación Sass
- Configurar Gulp en el package.json para habilitar la compilación de los estilos scss con Sass
- Inicio trabajo con la barra de navegación superior, se crea archivo _header.scss para iniciar a codificar estilos
- Inicio trabajo con el video que acompaña la parte inferior del header, se crea archivo _video.scss  para iniciar a codificar estilos
- Inicio trabajo con el video que acompaña la parte inferior del header, se crea archivo _sobre-mi.scss  para iniciar a codificar estilos
